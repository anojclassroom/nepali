
# Kodi Setup – PVR IPTV Simple Client

1) **Enable PVR**  
   Settings → Add‑ons → My add‑ons → *PVR clients* → **PVR IPTV Simple Client** → **Enable**.

2) **Load the playlist**  
   Configure → General → Location: *Local path* → M3U playlist path: `playlists/nepali_clean_kodi.m3u` → OK.

3) **Refresh**  
   Restart Kodi OR Settings → PVR & Live TV → Clear data → Confirm → Restart.

If any channel fails to play, it may require headers, be geo‑limited, or have changed endpoints. Replace with a fresh working link and reload.
